Commande Git

Git utilise un terminal de commande ou il est obligatoir de naviger via les commande git. En voici.

  • git --version -> Permet de connaitre la version de git (mise a jour, instalation correcte..)
  • git config --+++++ -> Formul de base pour la configuration de git
  • git config --list -> permet de voir la configuration
  • git config --global user.name "xxxx" -> configure le nom
  • git config --global user.email "xxxx" -> configure votre email
  • mkdir xxxx -> crée un nouvaux dossier projet. Normalement dans c/user/xxx(utilisateur)
  • cd xxxx -> Permet a git de rentrer dans un projet. Normalement dans c/user/xxx(utilisateur)
  • git init -> initialise un projet pour que ce soit bien un projet "git"
  • git statues -> Permet de connaitre les branches du projet.
  • git add xxxxx -> permet d'ajouté un ficher dans le dossier projet pour etre reconnu part git
  • git commit -m "xxxxxxx" -> permet de prendre la "photo" du projet/fichier. Le xxx est le commentaire.
  • git log -> donne un historique des modification du projet
  • git branch xxxxxx -> crée une nouvelle branche dans le projet
  • git branch -> voir les branches. La branche ou ont est auras un * et une autre couleur
  • git checkout xxxxx -> permet de changer de branche (les fichier retourne a l'étas d'avant chagement de branche)
  • git merge xxxxxx -> Permet de rapatrier une branche vers une autre branche (celle ou ont se situ)
